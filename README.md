# Solidity Security Research Skills

AI-powered Solidity security review skills for deep codebase analysis, bug hunting, false-positive reduction, exploit verification, and severity assessment.

## Supported Usage

Install this skills repository and run a full review on the codebase.

Install this skills repository and run the architecture reader on the codebase.

Install this skills repository and run the bug hunter on the codebase.

Install this skills repository and run the skeptic on all candidate findings.

Install this skills repository and run the exploit verifier on the reported issues.

Update skills to the latest version.

## Skills

- architecture-reader — maps protocol design, trust boundaries, privileges, value flow, and invariants
- bug-hunter — finds concrete vulnerability hypotheses
- skeptic — challenges findings and removes false positives
- exploit-verifier — verifies attack paths and realistic exploitability
- severity-triager — assigns realistic severity and confidence
- lead-auditor — produces the final structured report
- full-review — runs the full multi-agent review workflow

## Goal

Prefer fewer, stronger, carefully verified findings over many weak guesses.
